# Images Directory

This directory contains images for the life coaching website.

## Required Images:

### Profile Image
- **oxana-profile.jpg**: Professional headshot of Oxana for the About section
  - Recommended size: 400x500px
  - Format: JPG or PNG
  - Should convey warmth, professionalism, and approachability

### Optional Images
You can add additional images such as:
- Background images for hero section
- Icons for services
- Gallery photos from coaching sessions (with client permission)
- Certification badges

## Image Optimization Tips
- Compress images to reduce file size
- Use appropriate formats (JPG for photos, PNG for graphics with transparency)
- Consider using WebP format for better compression
- Ensure images are optimized for web (72 DPI)
